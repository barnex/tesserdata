# tesserdata

Extracting statistics from a game of tesseract.

## tesserstamp

Records frag time stamps form tesseract stdout. Usage:
``` 	tesseract | tesserstamp > timestamps.txt  ```
Output: time stamp in ms, boolean indication if you got fragged.
